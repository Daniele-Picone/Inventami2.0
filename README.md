# WineCellar

Progetto Next.js elaborato dai componenti inviati.

## Avvio

```bash
npm install
npm run dev
```

Apri `http://localhost:3000`.

## Route principali

- `/landing`
- `/sign-up-login-screen`
- `/dashboard`
- `/magazzino`
- `/digital-wine-menu`
- `/reports`
- `/users`
